# 🌊 Rising Waters – A Machine Learning Approach to Flood Prediction Rising Waters is a Machine Learning-based web application developed to predict flood risk using meteorological parameters such as rainfall, river discharge, humidity, soil moisture, and temperature. The project combines an end-to-end machine learning pipeline with an interactive Flask web application to assist users in assessing potential flood risks. --- ## 📌 Project Overview The application enables users to enter weather-related parameters through an intuitive web interface and receive an instant flood risk prediction. Multiple machine learning algorithms were trained and evaluated, with the best-performing model selected for deployment within the application. The system also provides prediction history, downloadable PDF reports, and visual insights into the model's performance. --- ## ✨ Features - 🌧️ Flood risk prediction using Machine Learning - 🤖 Comparison of four machine learning algorithms: - Decision Tree - Random Forest - K-Nearest Neighbors (KNN) - XGBoost - 📊 Automatic data preprocessing using StandardScaler - 🌐 Interactive Flask web interface - 🎨 Dynamic UI themes based on predicted flood risk level - 🗄️ Prediction history stored using SQLite - 📄 Downloadable PDF prediction reports - 📁 CSV export of prediction history - ⚡ Predefined weather scenarios for quick testing - 📈 Data visualization and model performance analysis --- ##
